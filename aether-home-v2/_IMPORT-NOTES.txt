FORMAT
------
"*.dc.html" are Claude Design *container* files. They use <x-dc>/<helmet>
templating and are rendered by support.js (a React runtime) inside the
Claude Design environment. Opening a .dc.html directly with a plain static
server / Live Server will NOT fully render it (support.js needs window.React
from the Design host). Use them as editable source; render/preview in
claude.ai/design. storymap.html renders standalone.
